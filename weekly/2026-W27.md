## Corrections this week (the supersede trail)

The week's highest-value content. Each was a user reframing that redirected
the investigation:

1. "Run dev15's passes, don't reconstruct from stock O3" → found the inliner.
2. "Soundness is the wrong first reflex" — the indvars fold is a *sound exact
   summary*; the regression was a lost flag. (Superseded the 06-29 entry.)
3. "Fix the test, not the pass" — making the test non-summarizable instead of
   keeping indvars off to please a degenerate test.
4. "coi without unify is unsupported" — killed a wrong "COI is unsound
   standalone" claim mid-investigation.
5. "The supported mode is df + coi + unify-assumes + gsa; that is durable" —
   the final framing: one 4-flag mode, not a bug in any single flag.

Pattern across all five: my first framing over-localized blame (a pass, a
flag, an analysis); the durable fact was each time about a *configuration* —
which pieces are meant to run together.
